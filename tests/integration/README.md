# Integration Tests

This directory contains integration tests for your api-service.

## Getting Started

1. Review `example.test.js` for patterns specific to api-service projects
2. Rename and customize the example test for your use case
3. Run integration tests: `npm run test:integration`

## Test Types for api-service


- **Unit Tests**: Individual functions and modules
- **Integration Tests**: Database operations, API endpoints
- **E2E Tests**: Full request/response cycles
- **Performance Tests**: Load testing, concurrency
